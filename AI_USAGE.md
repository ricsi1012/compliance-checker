# AI Használat Dokumentáció

## Használt eszközök
- **ChatGPT / Claude:** Backend boilerplate kódok (Java/C#) generálása, Docker konfiguráció hibakeresése.
- **GitHub Copilot:** React komponensek gyors kódolása és Tailwind CSS formázás.

## Konkrét prompt példák
1. *"Create a .NET 8 Web API with a generic AIService class that connects to OpenAI API."*
2. *"Generate a Dockerfile for a multi-stage Maven build for Spring Boot 3."*
3. *"Write a comprehensive ISO 27001 Security Policy document text that satisfies requirements like MFA, RBAC, and Quarterly Reviews."*

## Időmegtakarítás
Az AI használata becslésem szerint kb. **3-4 óra tiszta kódolási időt spórolt meg**, különösen a Docker konfiguráció és a mintaadatok generálása terén.
